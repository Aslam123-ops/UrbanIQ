# 🚀 UrbanIQ – Smart City AI

UrbanIQ is an AI-powered system designed to improve decision-making in smart cities by analyzing data and providing real-time recommendations.

---

## 🌍 Problem
Modern cities face:
- Traffic congestion
- Energy inefficiency
- Slow decision-making

---

## 💡 Solution
UrbanIQ uses Machine Learning to:
- Analyze traffic data
- Predict energy usage
- Provide smart recommendations

👉 Data → AI → Smart Decisions

---

## 🧠 Features
- 📊 Traffic analysis
- ⚡ Energy prediction
- 🤖 AI-based recommendations
- 🖥 Interactive dashboard (Streamlit)

---

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

---

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
